# Sistema de Gestión y Control para una Papelería

## Descripción

Este proyecto consiste en una aplicación de escritorio desarrollada en JavaFX que permite administrar una papelería mediante el control de usuarios, productos, categorías, proveedores, ventas e inventario. El sistema implementa los principios de la Programación Orientada a Objetos y utiliza una base de datos MySQL alojada en Railway.
